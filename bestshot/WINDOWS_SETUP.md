# Windows環境でのセキュリティ警告（Smart App Control）の回避方法

Windows 11では、「Smart App Control (スマート アプリ コントロール)」や「SmartScreen」によって、デジタル署名がされていない（または信頼性が確立されていない）新しい実行ファイルが警告・ブロックされて起動できない場合があります。

これを安全に回避してアプリを起動するための手順を以下に示します。

---

## 対策 A：自己署名証明書付き MSIX インストーラーの使用（推奨）

このプロジェクトでは、ストアアプリ形式のインストーラー（`.msix`）をビルドしてインストールする方法をサポートしています。自己署名証明書をシステムに一度信頼させることで、警告なしに起動できるようになります。

### 1. MSIXパッケージのビルド
プロジェクトのルートディレクトリで以下のコマンドを実行します。
```bash
flutter build windows
flutter pub run msix:create
```
※ コマンド実行に成功すると、`build/windows/x64/runner/Release/bestshot.msix` にインストーラーが生成されます。また、同じフォルダに `bestshot.pfx` などの証明書ファイルが生成される場合があります。

### 2. 自己署名証明書の信頼手順
`.msix` ファイルをそのままダブルクリックすると「証明書が信頼されていません」という警告が出てインストールできない場合があります。その場合は以下の手順で証明書を信頼させます。

1. 生成された `.msix` ファイル（または `.pfx` 証明書ファイル）を右クリックし、**「プロパティ」** を開きます。
2. **「デジタル署名」** タブを開き、署名リストにある項目を選択して **「詳細」** ボタンをクリックします。
3. **「証明書の表示」** をクリックし、**「証明書のインストール」** をクリックします。
4. 保存場所に **「ローカル コンピューター」** を選択し、**「次へ」** をクリックします（管理者権限が必要）。
5. **「証明書をすべて次のストアに配置する」** を選択し、**「参照」** ボタンをクリックします。
6. リストから **「信頼されたルート証明機関」** または **「信頼された発行元」** を選択して **「OK」** をクリックします。
7. インストールを完了させます。

証明書のインポート完了後、再度 `.msix` ファイルをダブルクリックすると、警告なしでインストール画面が立ち上がり、安全に起動できるようになります。

---

## 対策 B：通常の実行ファイル（.exe）のブロック解除手順

`bestshot.exe` を直接ビルドして実行する際、Smart App ControlやSmartScreenでブロックされた場合は、以下の手順でブロックを解除できます。

1. ビルドされた実行ファイル `build/windows/x64/runner/Release/bestshot.exe` を右クリックし、**「プロパティ」** を開きます。
2. 全般タブの最下部にあるセキュリティ項目で、**「許可する」**（または「ブロックの解除」）チェックボックスにチェックを入れます。
3. **「適用」** ➔ **「OK」** をクリックします。

これにより、OSはそのファイルを「ユーザーが意図して実行を許可した安全なファイル」と見なし、Smart App Controlによる起動ブロックが解除されます。

---

## 対策 C：Microsoft Defender への安全性の申請（開発者向け）

広く一般に配布する場合は、以下のMicrosoft Security Intelligenceポータルから、ビルドしたファイルを「誤検知 (False Positive)」としてサンプル送信（Submit）します。

*   **送信先ポータル**: [Microsoft Security Intelligence - Submit a file](https://www.microsoft.com/wdsi/filesubmission)
*   **手順**:
    1. 「Developer (開発者)」としてサインインします。
    2. ビルドした `bestshot.exe` と `opencv_dart.dll` をZIP等にまとめてアップロードします。
    3. 解析完了後、Microsoftの定義データベースが更新され、世界中のWindows端末で Smart App Control / SmartScreen のブロックが自動的に解除されます（通常1〜3日程度）。
