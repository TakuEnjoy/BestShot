# BestShot (ベストショット写真整理アプリ)

一眼レフやミラーレスカメラ等で撮影された大量の連写・類似写真の中から、被写体のピントや表情（顔・目閉じなど）を OpenCV と機械学習を用いて自動判定し、最も優れた「Best」写真を提案・選別して不要な類似写真を高速に整理する **iOS / iPadOS 専用**アプリです。

---

## 🌟 主な機能と仕様

### 1. 写真のインポート & 高速バックグラウンド解析
* **選べるインポート方法**:
  * **写真ライブラリからインポート**: iPad/iPhoneのネイティブ写真ピッカー（ギャラリー）を使い、対象の写真を直接選択してインポートします。
  * **ファイルからインポート**: 外部ストレージやフォルダから画像をインポートします。最適化された $O(N)$ アルゴリズムによる高速な RAW プレビューJPEG抽出により、巨大なRAWファイルも即座に処理します。
* **調整可能な上限枚数**: インポート上限件数は、撮影量や端末スペックに合わせて **50枚 / 100枚 / 200枚 / 500枚 / 1000枚** からUIドロップダウンでいつでも調整可能。
* **高速マルチスレッド解析 (Isolate)**:
  * **類似判定**: pHash（Perceptual Hash）アルゴリズムにより、構図の類似度を高速計算。
  * **ピント（鮮明度）判定**: Laplacian Variance（ラプラシアン分散）を用いてシャープネスを評価。
  * **露出評価**: 輝度ヒストグラムから、白飛び・黒潰れ（クリッピング）をスコア化。
  * **特徴点抽出**: ORB特徴点記述子を計算し、類似構図のグループ化に利用。
* **ポートレート（顔優先）モード**:
  * 顔が検出された写真では、顔および瞳の領域（ROI）のピント精度を個別に高精度評価します。
  * 平均目の開き度合いを数値化し、「目閉じ」を自動検知してスコアを大幅減点（Best除外）します。
  * Google ML Kit を使用して顔・瞳を高精度に検出します。

### 2. 時系列と類似度（色彩加味）に基づく自動グループ化
* 指定された「連写時間（1秒〜60分）」および「画像の類似度（色彩・特徴点・pHash）」に基づいて、類似する一連の写真群を自動的にグループ分けします。
* **適応型HSVカラー判定**: 
  * 画像ごとの色彩分布から動的にしきい値を計算する「適応型HSVマスク（Adaptive HSV Masking）」により、白い壁などの無彩色背景を除去し被写体の色のみを正確に抽出。
  * 鮮やかな写真では色相（Hue）の差異を最重視し、無彩色写真では彩度（S）・明度（V）に自動シフトして比較する「適応的加重平均（Bhattacharyya距離）」を採用。これにより「白い壁の前の緑の被写体」などの特殊な状況も含め、あらゆる写真で過剰分離や誤分類を起こさない極めて頑健な色彩類似度判定を実現しています。
* グループごとに、鮮明度や顔の状態が最も良いと判定された写真に **「Best」バッジ** が自動付与されます。

### 3. 洗練された整理・仕分けUI (プレミアムUI/UX)
* **「Best以外を削除候補に」の一括操作**: グループヘッダーのボタンから、ベストショット以外の類似写真を1タップで一括して削除候補（ゴミ箱行き）に指定可能。
* **削除候補のフェードアウト表示**: 削除チェックを入れた写真は、サムネイルが **モノクロ＋半透明（不透明度 0.45）** に変化し、残す写真と消す写真が画面全体で直感的に識別可能。
* **「Best」写真の視覚強調**: 自動/手動でベストショットに設定された写真は、周囲が **グリーンの太い枠線と柔らかな光彩シャドウ (boxShadow)** で包まれ、視覚的に際立ちます。

### 4. 2画面〜4画面の比較プレビュー（ルーペモード）
* **元解像度（フル解像度）による超精細フォーカスマスク**:
  * ピントが合っている箇所を白くハイライトする「フォーカスマスク」機能を搭載。
  * リサイズによるエッジ情報の劣化を防ぐため、ルーペ時は**元解像度のまま高精細にピント検出**を行い、まつ毛や肌の微細な質感をドットバイドットで正確に可視化します。
  * 暗所や高感度撮影のデジタルノイズを無視する「ノイズ耐性加重比率」に調整済み。
  * マスク描画色は、被写体や背景色に合わせて **「白 ➔ 赤 ➔ 黄 ➔ 緑」** に切り替え可能。
* **高速セッションキャッシュ**:
  * ルーペ画面で一度生成された高解像度マスクは、アプリ実行セッション中に自動でメモリ上にキャッシュされます。ルーペ画面を閉じたり切り替えたりしても、**再計算なし（0ms）で即時表示** されます。
* **縦横比を維持した正確なプレビュー表示**:
  * 縦横比（アスペクト比）を正確に保持したまま、画面いっぱいに無駄なく表示（BoxFit.contain）するため、写真が不自然に引き伸ばされることはありません。
* **プレビューしたままその場で仕分け**:
  * 各ペイン下部の半透明フローティングバーから、拡大確認しながら直接「削除候補のチェック」や「Bestへの指定（スターボタン）」が可能です。
* **洗練されたヒストグラム表示**:
  * 輝度分布を、アプリのテーマカラーの滑らかなグラデーションで美しく描画するガラスモルフィズム風ヒストグラムパネルを配置しています。

## 🛠 技術スタック

* **フレームワーク**: Flutter (Dart)
* **画像処理ライブラリ**: 
  * `opencv_dart`: OpenCV C++ バインディング（ピント検出、モルフォロジー演算、二値化、Cannyエッジ）
  * `image`: RAWプレビューJPEG抽出、およびフォールバック用のピント演算
* **機械学習ライブラリ**: `google_mlkit_face_detection` (高精度顔・表情解析)
* **画像比較ライブラリ**: `image_compare` (pHash類似度判定)
* **メタデータ解析**: `exif` (シャッタースピード、F値、ISO、撮影日時などのEXIF情報取得)
